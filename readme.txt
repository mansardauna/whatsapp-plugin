Turn your website visitors into WhatsApp leads automatically. Tracks clicks, captures visitor data, and provides a simple CRM inside your WordPress dashboard.

== Description ==

WP WhatsApp CRM is a lightweight yet powerful plugin designed to help you capture and manage WhatsApp leads directly from your website. With a stylish floating button and a built-in CRM dashboard, you can monitor exactly which pages are driving your WhatsApp conversations.

**Key Features:**
* **Floating WhatsApp Button**: High-performance, stylish button that opens a pre-filled chat.
* **Lead Capture**: Automatically saves the user's phone, page URL, IP, and timestamp when they click.
* **CRM Dashboard**: View and manage all your leads in one place.
* **Export to CSV**: Download your leads data for further analysis or CRM integration.
* **Configuration Summary**: See your active WhatsApp settings directly on the leads table.
* **Pro Ready**: Built with an upgrade path for WhatsApp Cloud API automation and advanced tagging.

== Installation ==

1. Upload the `wp-whatsapp-crm` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **WhatsApp CRM > Settings** to configure your phone number and message.

== Frequently Asked Questions ==

= Is this plugin mobile-friendly? =
Yes, the floating button is fully responsive and works perfectly on mobile devices.

= Where are the leads stored? =
Leads are stored in a custom database table (`wp_wwc_leads`) to ensure performance and easy data management.

= Can I export my leads? =
Absolutely. There is an "Export to CSV" button available on the Leads Dashboard.
